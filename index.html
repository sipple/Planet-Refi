Planet-Refi by saalon

A mortgage refinance timeline for the web developed for Planet Money

This project allows contributors to enter events that will then be displayed using a Simile timeline widget. This was built to track the refinance process of 3 Planet Money users.

Dependencies

The project uses PHP 5, MySQL & Javascript, as well as linking to the Simile Timeline widget.
